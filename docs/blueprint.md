# **App Name**: HealthSight AI

## Core Features:

- AI Skin/Wound Analysis: Use the device's camera and on-device models to analyze skin conditions and wounds.
- AR Anatomy Visualizer: Use AR to visualize anatomy for patient education. Supports ARKit and ARCore.
- Secure Patient Data Storage: Store and manage patient data securely using Cloud Firestore, ensuring compliance with privacy standards.
- AI-Powered Symptom Checker: AI chatbot that suggests possible conditions based on user-provided symptoms using OpenAI API tool for medical knowledge.
- Medication Reminders: Provide medication reminders based on prescriptions stored in Cloud Firestore.
- Remote Patient Monitoring: Collect and display real-time health data from wearable devices (Fitbit, Apple HealthKit).
- Generative AI Report Summaries: Use generative AI tools to create reports summarizing findings of images/vidoes

## Style Guidelines:

- Primary color: Deep teal (#008080) to represent trustworthiness, health, and a calming user experience.
- Background color: Very light teal (#E0F8F7) for a clean, modern feel.
- Accent color: Muted blue-green (#468479) for highlights and interactive elements.
- Body and headline font: 'Inter', a sans-serif font, is used to create a machined, neutral, and highly readable user interface. It works well for both headings and body text.
- Use clear and intuitive icons to represent health metrics, functions, and categories.
- Employ a clean, card-based layout to display data. Each data point and functionality should be clearly separated and easy to find.
- Subtle transitions and animations on data updates and user interactions to enhance user engagement.
